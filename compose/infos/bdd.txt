
 -Tables-

User : 
	- id (key)
	- Nom (varchar)
	- prenom (varchar)
	- description (varchar)
	- age (integer)

Projet : 
	- id (key)
	- nom_projet (varchar)
	- description (varchar)

Owner :
    – id_projet (clé étrangère vers projet)
    - id_user (clé étrangère vers user)

projet_lien :
	- id_projet (clé étrangère vers projet)
	- id_lien (clé étrangère vers lien)

projet_tag :
	- id_projet (clé étrangère vers projet)
	- id_tag (clé étrangère vers tag)

Lien : 
	- id (key)
	- lien_http (varchar ou autre)
	- type (varchar)

Tag :
	- id (key)
	- labelle (varchar)
